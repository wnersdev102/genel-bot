## 📦 Kurulum

**1.** [`ayarlar.json`](https://github.com/wnersdev102/asiri-gelismis-genel-bot/blob/main/ayarlar.json) dosyasını tamamen doldurun

**2.** Modülleri indirin:
```bash
npm i
```

**3.** Botu başlatın:
```bash
node wnersdev.js
```
